# Combine a fixed and random effects formula

Combine a single fixed and random effects formula by pasting them
together.

## Usage

``` r
combine_formulas(fixed, random)
```

## Arguments

- fixed:

  fixed effects formula (two-sided `formula` object)

- random:

  random effects formula (one-sided `formula` object)

## Details

Internal function, used in
[`combine_formula_lists()`](combine_formula_lists.md).
