# Choose default analysis model based on outcome and data level

Choose default analysis model based on outcome and data level

## Usage

``` r
choose_default_model(outcome, lvl, max_lvl)
```

## Arguments

- outcome:

  outcome variable

- lvl:

  hierarchical level of the outcome variable

- max_lvl:

  maximum hierarchical level in the data

## Value

character string giving the default model type
