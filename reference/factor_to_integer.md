# Convert a factor to an integer representation

Convert a factor to an integer representation

## Usage

``` r
factor_to_integer(x)
```

## Arguments

- x:

  a vector. If `x` is a factor it will be converted to integers as
  described below; otherwise `x` is returned unchanged.

## Value

An integer vector when `x` is a factor, otherwise the original `x`.

## Note

Used in `extract_outcome_data()`.
